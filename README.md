## 안내

본 저장소는 저작자가 개인적으로 사용하기 위해 제작·관리하고 있는
프로젝트입니다. 저장소가 공개(public)로 설정되어 있는 것은 관리상의
편의를 위한 것일 뿐, 외부 사용을 위한 공개 배포가 아닙니다.

본 저장소의 모든 코드, 문서, 리소스에 대하여 다음 행위는 허용되지
않습니다:

- 다운로드 및 실행
- 복제, 수정, 재배포
- 상업적·비상업적 목적의 일체의 사용
- 본 코드를 기반으로 한 파생 저작물 작성
